# Node Chat App
A demo app I intend to set up on a cloudflare worked.

It is a small test demo of a web socket using chat client.

Very much a WIP rn.

## Project Setup

```sh
#install node then
cd ./nodechatclient
npm i

cd ../.nodechatserver
npm i

# run server
nodemon chatserver.js
```

In another window open a terminal in nodechatclient
```sh
npm run dev
```
